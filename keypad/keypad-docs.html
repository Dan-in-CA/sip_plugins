About the Keypad Plugin

This plugin has been created to provide a simple user input for SIP. A 4X4 scanning keypad is used to handle user input. A total of 8 GPIO pins are used on the Raspberry Pi for this plugin. Using this keypad, functionality such as manually running a station can be done from the hardware.

Installing the Hardware

These instructions assume that you are using B+, version 2, version 3, or pi zero hardware. To use other models of the Raspberry Pi, keypad.py may need to be modified to do so.

A 4X4 scanning keypad is needed. This plugin assumes that the keypad has the following key layout, but the layout may be modified within the keypad.py file.

Keypad Keys

The letter keys A through D are used to select different functions. A momentary press of one of these keys will select a value entry function. The plugin will then wait for a value to be entered followed by pressing the pound key (#). The asterisk key (*) may be pressed at any time to cancel an entered value. Pressing and holding one of the function keys will immediately execute a function that does not require a value to be entered.

Buzzer

This keypad plugin uses the buzzer plugin when it is installed and enabled. The buzzer is used to provide audio feedback when keys are pressed and functions are executed.

The buzzer profiles may be changed from the keypad settings page. By default, the following beeps are used.

Click (25 ms beep): Used to acknowledge button press

One short beep followed by a long beep: Entry invalid or cancel

One short beep: Number value entry is acknowledged

Two short beeps: Function has been executed

Using the Plugin

Function Keys

The function keys A through D may be assigned from the keypad settings page.  The group labeled “Function Keys” contain functions that require a value to be entered. By default, “Start Manual Station” is assigned for A. Pressing the A key on the keypad will set the plugin to accept values to start a station manually. For example, keying in A-2-# will stop any running program and start station 2 immediately for the number of seconds entered for “Manual Station Time”. Then keying in 1-3-# will stop station 2 and start station 13. Entering 0-# will immediately stop all stations. Below is a list of selectable functions.

·         Start Manual Station (Default to A key)

o   Entered value will stop any program running and immediately run a single station

·         Start Manual Program (Default to B key)

o   Entered value will stop any program running and immediately start the entered program number

·         Set Water Level (Default to C key)

o   Entered value sets the water level value as a percentage

·         Set Manual Station Time (Default to D key)

o   Entered value sets the number of seconds for manual station time function

·         Set Rain Delay Time

o   Entered value sets the number of hours for rain delay hold function (explained below)

Press and Hold Functions

The group labeled “Press and Hold Functions” on the settings page contain single-shot executable functions that do not require a subsequent value to be entered. By default, no hold functions are assigned to any function key. If, for example, you were to set “Stop All Stations” to the A key, pressing and holding the A key will immediately stop any running program. Below is a list of selectable functions.

·         Stop All Stations

o   Immediately stops any running program

·         Activate Rain Delay

o   Sets the rain delay for the number of hours entered for “Rain Delay Activation Time”

·         Deactivate Rain Delay

o   Deactivates rain delay

·         System On

o   Enables the system

·         System Off

o   Disables the system

·         Restart System

o   Restarts the system

·         Reboot OS

o   Reboots the Raspberry Pi
