MCP Server Plugin

This plugin exposes an MCP Streamable HTTP endpoint for SIP control and status.

Endpoint

/mcp

Authentication

  • SIP admin session cookie, or
  • X-SIP-Passphrase header, or
  • Authorization: Bearer <passphrase>

Tools

  • get_system_status
  • list_stations
  • list_programs
  • run_station
  • stop_all_stations
  • run_program_now
  • set_rain_delay
  • set_water_level
  • get_logs
