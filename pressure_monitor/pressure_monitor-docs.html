Pressure Monitor Plugin Documentation

About the Pressure Monitor Plugin

Displays a graph of historical water pressure at the bottom of the schedule timeline.

This requires a water pressure sensor, hooked up through Arduino and made available through serial communication. Many aspects are hard wired in such as max PSI - it may require modification depending on the hardware. This is not intended to be a general-purpose solution.

Using the Pressure Monitor plugin

Settings include the port of the Arduino which is reading data from the pressure sensor, and a Remote URL. This allows you to fetch the SVG from an alternate RPi (that also has this plugin installed) facilitating monitoring pressure from a single source that is feeding into multiple SIP systems (I have 3 systems fed by a single pump.)
